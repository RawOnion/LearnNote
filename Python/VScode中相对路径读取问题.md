## VScode中相对路径与一般的编辑器差异

|   符号    |                 vscode                  |           一般编辑器            |
| :-------: | :-------------------------------------: | :-----------------------------: |
| `./2.txt` | 相对于工作路径(`.vscode`文件夹所在位置) |       相对于当前运行文件        |
|  `2.txt`  |             想读与工作路径              |       相对于当前运行文件        |
| `/2.txt`  |     相对于磁盘根目录(windows系统下)     | 相对于磁盘根目录(windows系统下) |
