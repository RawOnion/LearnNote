# Linux命令格式：

`command [options] [arguments]`，其中`options`以`-`开始，多个选项可用一个`-`连起来，如`ls -l -a`与`ls -la`效果相同。

`command`:命令

`options`:`--单词`或`-单个英文字符`



# `[],<>,|`等符号的含义：

1. `[]`表示可选的；
2. `<>`表示是可变化的；
3. `x|y|z`表示选择一个；
4. `-abc`表示三个参数(或任何两个)的混合使用。
